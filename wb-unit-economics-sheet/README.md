# Юнит-экономика WB — Google Таблица

Готовый файл **без Apps Script**. Загружаете в Google Drive — получаете таблицу с формулами.

## Файлы

| Файл | Когда использовать |
|------|-------------------|
| **`Юнитка-WB-Google.xlsx`** | Загрузка в **Google Таблицы** (формулы с `;`, INDEX/MATCH) |
| `Юнитка-WB.xlsx` | Excel / Windows |

> Если в таблице много `#ОШИБКА!` / `#ERROR!` — вы загрузили старую версию. Используйте **Юнитка-WB-Google.xlsx**.

Внутри:
- **Юнитка** — расчёт прибыли FBO/FBS, маржи, логистики (499 строк с формулами)
- **_Настройки** — УСН 11%, комиссия 1.75%, тарифы WB
- **_Цены_закупки** — 501 закупочная цена из вашей таблицы
- **_Комиссия_ВБ** — 631 комиссия FBO/FBS по артикулу

## Как открыть в Google Таблицах (2 минуты)

1. Откройте [Google Drive](https://drive.google.com)
2. **Создать → Загрузить файл** → выберите **`Юнитка-WB-Google.xlsx`**
3. После загрузки: правый клик по файлу → **Открыть с помощью → Google Таблицы**
4. Google создаст копию в формате таблицы — формулы сохранятся

Рекомендуется: **Файл → Создать копию** (чтобы работать в нативном формате Google, а не в режиме совместимости с Excel).

## Что заполнять

В листе **Юнитка** для каждого товара:

| Колонка | Что ввести |
|---------|------------|
| B | Артикул WB |
| C | Артикул продавца (подтянет закупку и комиссию) |
| L | Цена продажи |
| T | Наша цена на WB |
| W, X, Y | Длина, ширина, высота (см) |
| AA | Коэфф. склада (или пусто → 2.2) |
| AM | Упаковка, ₽ (или пусто → 65) |
| Q | Целевая рентабельность FBO, % (для расчёта цены в O) |

Остальное считается автоматически.

## Формулы (как в вашей таблице)

- **Прибыль FBO** = продажа − закупка − комиссия FBO − налог − упаковка − брак − доставка с возвратом
- **Маржа** = прибыль / цена продажи
- **Логистика** = (46 + max(0, объём−1) × 14) × коэфф. склада; возврат × 1.0454

## Автоподгрузка с WB API (без Apps Script)

Всё подтягивается автоматически, **кроме закупочной цены** (колонка K).

| WB API | Данные в таблице |
|--------|------------------|
| Карточки | артикул, бренд, название, габариты |
| Цены | продажа, базовая, наша цена |
| Маркетплейс | остатки FBS |
| Статистика | остатки FBO |
| Аналитика | заказы за 7 дней |
| Тарифы | комиссии FBO/FBS, логистика |

### Тест (только JSON)

```bash
npm run sync:unit:json
```

### Запись в Google Таблицу

1. [Google Cloud](https://console.cloud.google.com) → проект → **Google Sheets API** → включить
2. Сервисный аккаунт → ключ JSON
3. Таблицу расшарить на email аккаунта (Редактор)
4. В `.env.local`:

```env
GOOGLE_SHEET_ID=ваш_id_таблицы
GOOGLE_SERVICE_ACCOUNT_PATH=/путь/к/key.json
WB_API_TOKEN=...
```

5. `npm run sync:unit` — закупки в колонке K не затираются
